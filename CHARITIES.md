## How It Works (User Flow)

1. User selects a charity category from the website
2. User is directed to the charity's donation page (external link)
3. After donating, user receives a Donation ID / receipt number
4. User enters the Donation ID on our website
5. Consultation form unlocks → user fills birth details
6. Reading is delivered within 24 hours

---

## Notes

- All listed organizations are registered NGOs with 80G tax exemption (India)
- International donors can use GlobalGiving or Give India as intermediaries
- Minimum donation amounts are intentionally low to be accessible
- The ₹108 amount is sacred in Hinduism (108 beads in a mala)
- Donation receipts serve as the "Good Karma ticket" to access consultation
